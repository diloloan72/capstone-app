## To run locally: 
```
npm start 
```


## To deploy to Heroku 
```
npm run build 
```
- Copy content of build folder to ../server/frontend 
- Redeploy Heroku


## To deploy to CPanel 
```
npm run build 
```
- zip contents of build folder
- copy zip file to `public_html` of the CPanel server 
- Unzip the file
